## Components used in this build:

## Microcontroller & Core
- **1x NodeMCU ESP8266** (CP2102 version)
  - WiFi-enabled microcontroller
  - ~$4-5 USD (400 INR)

## Display & Interface
- **1x 0.96" OLED Display (128x64)**
  - I2C interface (SSD1306 driver)
  - ~$5 USD (450)

- **3x 6x6mm Tactile Push Buttons**
  - For menu navigation (Up, Down, Select)
  - ~$0.10 each (10 INR)

## Power System
- **1x 18650 Li-ion Battery** (3.7V, 2500-3000mAh)
  - ~$3-5 USD (270 INR)

- **1x TP4056 Charging Module**
  - USB-C battery charging circuit
  - Over-charge/discharge protection
  - ~$1 USD (60 INR)

## RF Components
- **1x 2.4GHz External Antenna** (3-5dBi)
  - U.FL/IPEX connector
  - ~$2-3 USD (280 INR)

- **1x U.FL to RP-SMA Pigtail Cable**
  - ~$1-2 USD (Included but ~100 INR)
